# Bena Media Helpers
(Tiếng Việt ở bên dưới)
## Overview (English)

Bena Media Helpers streamlines common media workflows on WordPress websites. The plugin can automatically pull remote images into your media library, remove attachments when posts are deleted, convert image uploads to WebP, and apply either image-based or text watermarks with live previews and fine-tuning options.

### Installation
1. Download the plugin archive or clone the repository into `wp-content/plugins/bena-media-helpers`.
2. If using a ZIP file, upload it via **Plugins → Add New → Upload Plugin**, or unzip manually into the plugins directory.
3. Activate **Bena Media Helpers** from the **Plugins** page in the WordPress dashboard.

### Key Features
- Save external images to the WordPress media library automatically.
- Apply a consistent filename prefix for new uploads.
- Delete orphaned attachments when their parent post is removed.
- Convert JPG/PNG uploads to WebP with adjustable compression.
- Enable watermarking with PNG overlays or custom text, color, size, position, and opacity.

### Usage
1. Navigate to **Settings → Bena Media Helpers** in the WordPress dashboard.
2. Toggle features in the “Post Image Settings” card to match your workflow.
3. In the “Watermark” card, choose image or text watermarking, upload assets or enter text, and adjust scale/opacity.
4. Click **Save settings** to apply your configuration; the plugin will enforce these rules for subsequent uploads.

## Tổng quan (Tiếng Việt)

Bena Media Helpers giúp tối ưu quy trình xử lý hình ảnh trên website WordPress. Plugin có thể tự động lưu ảnh ngoài về thư viện, xóa ảnh đính kèm khi xóa bài, chuyển ảnh tải lên sang WebP và đóng dấu watermark bằng ảnh PNG hoặc chữ với nhiều tùy chỉnh trực quan.

### Cài đặt
1. Tải plugin (file ZIP) hoặc clone repo vào thư mục `wp-content/plugins/bena-media-helpers`.
2. Nếu dùng file ZIP, vào **Plugins → Add New → Upload Plugin** để tải lên, hoặc giải nén thủ công vào thư mục plugins.
3. Kích hoạt **Bena Media Helpers** tại trang **Plugins** trong bảng điều khiển WordPress.

### Tính năng chính
- Tự động lưu ảnh ngoài vào thư viện WordPress.
- Thêm tiền tố cố định cho tên file khi tải ảnh.
- Xóa ảnh đính kèm khi bài viết bị xóa để tránh rác.
- Chuyển đổi ảnh JPG/PNG sang WebP với mức nén tùy chỉnh.
- Đóng watermark bằng ảnh PNG hoặc chữ, điều chỉnh màu, kích thước, vị trí và độ trong suốt.

### Cách sử dụng
1. Vào **Cài đặt → Bena Media Helpers** trong trang quản trị WordPress.
2. Bật/tắt các tính năng trong mục “Post Image Settings” theo nhu cầu.
3. Ở mục “Watermark”, chọn dạng watermark bằng ảnh hoặc chữ, tải lên ảnh hoặc nhập nội dung và tinh chỉnh thông số.
4. Nhấn **Save settings** để lưu cấu hình; plugin sẽ áp dụng cho những lần xử lý hình ảnh tiếp theo.
